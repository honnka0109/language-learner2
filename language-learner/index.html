📋 사용 방법

  1. iPhone으로 NFC 태그에 가까이 대세요
  2. 자동으로 해당 Scene 페이지가 열립니다
  3. 성우 음성을 듣고 따라 말해보세요
  4. 녹음 버튼으로 내 목소리를 저장하세요
  5. 반복 학습으로 실력을 향상시키세요!
